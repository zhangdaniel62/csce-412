Detailed Description

Program entry point for the load balancer simulation.

This file initializes logging (via spdlog), loads a YAML configuration, constructs the firewall (IPBlocker) and LoadBalancer, and runs the simulation for the configured number of clock cycles.

Logging: A shared spdlog logger named "lb" is configured with both a colored console sink and a file sink (logs/loadbalancer.txt).

void init_logging ( )

Initialize the shared simulation logger.

Creates a spdlog logger named "lb" with two sinks:

  • A colored console sink for terminal output.
  • A file sink that appends to logs/loadbalancer.txt.

If the logger already exists, this function does nothing.
