Detailed Description

Simulates a single server that processes one request at a time.

Each WebServer maintains a busy/idle state and advances its assigned request in discrete clock cycles via processCycle().

bool WebServer::processCycle ( )

Advance processing by one clock cycle.

Process one clock cycle of work.

If the server is busy, the request's remaining time is decremented. If the request completes, the server becomes idle.

Returns
true If the server was busy this cycle.
false If the server was idle.

If the server is busy, the remaining time of the active request is decremented by one. If the request completes, the server transitions back to the idle state.
