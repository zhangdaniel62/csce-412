Detailed Description

Manages request distribution, server processing, scaling, and statistics.

The load balancer maintains a FIFO queue of incoming requests, assigns them to available servers, and processes requests over time. A cooldown timer prevents scaling decisions from happening too frequently.

void LoadBalancer::addRequest ( Request req)

Add a request to the internal queue if not blocked.

Enqueue a request if it is not blocked by the firewall.

The request's source IP is checked by the IPBlocker. If blocked, the request is discarded and counted as blocked; otherwise it is enqueued.

Parameters
reqRequest to enqueue (moved into the queue on success).

Requests with a blocked source IP are discarded and counted as blocked. Accepted requests are appended to the internal FIFO queue.
