# CSCE 412 Project 3: Load Balancer Simulation

## Compiling and Running the Simulation

> \[!NOTE]
> This project requires a C++23 compiler, and requires cmake version 3.16 or newer.

Run the following commands to compile and run the simulation:

```bash
make run
```

Run the following commands to clean up the simulation:

```bash
make clean
```

Run the following command to run the simulation:

```bash
make run
```
